# Joining a Wi-Fi Network

## Overview
A device joins Wi-Fi by finding a network, entering credentials, getting network settings, and testing the connection.

## Step-by-step explanation
1. The device scans for Wi-Fi networks.
2. It finds a network, or the user selects one.
3. The user enters the password or other credentials.
4. The network checks the credentials and allows access.
5. DHCP gives the device an IP address and other settings.
6. The device can now reach the network and internet.
7. The device tests the connection.

## Security or failure points
- An open network may not require a password.
- Weak security can allow unauthorized access.
- A weak signal can break the connection.
